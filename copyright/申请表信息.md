## 五、软件用途和技术特点

### 软件用途

田野笔记管理工具软件是一款专为语言学研究者设计的桌面应用程序，主要用于田野调查过程中的语料数据采集、管理和分析。软件面向从事方言调查、少数民族语言记录、语言接触研究等领域的科研工作者，解决了他们在田野调查中需要高效管理带有国际音标（IPA）和语法标注（Leipzig Glossing Rules）的语料数据的核心需求。用户可通过本软件完成语料录入、检索筛选、格式化导出以及 AI 辅助分析等全流程工作，显著提升语言学田野调查数据处理的效率和规范性。

### 技术特点

本软件采用 Python 语言开发，基于 PyQt6 图形界面框架构建跨平台桌面应用，支持 macOS、Windows 和 Linux 三大主流操作系统。数据存储层使用 SQLite 嵌入式数据库，采用自动迁移机制实现数据库 schema 的版本化管理，并通过索引优化提升大数据量下的查询性能。

软件的核心创新点包括：（1）针对语言学语料的专业化数据模型设计，支持单词、单句、语篇、对话四种数据类型，通过分组机制关联语篇和对话的多条记录；（2）集成 IPA 国际音标快速输入面板，支持 Unicode 和 IPA 字符的完整输入与显示；（3）Word 文档导出引擎实现了符合 Leipzig Glossing Rules 的词对词自动对齐排版算法，基于累计宽度检测的动态分行策略确保长句在页面中正确换行；（4）多 LLM 混合架构的 AI 辅助功能，支持 Claude、OpenAI 兼容协议和 Ollama 本地模型三种接入方式，通过 Provider 抽象层和自动切换机制为用户提供灵活的 AI 辅助词汇分解和翻译服务；（5）API 密钥通过操作系统 keyring 服务安全存储，避免明文泄露风险。
